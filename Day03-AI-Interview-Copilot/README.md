## ✨ Features

### 👔 Recruiter Workspace

* Resume Analysis
* ATS Compatibility Score
* AI Semantic Matching
* Risk Analysis
* Hiring Recommendation
* Resume Ranking

### 🎯 Candidate Workspace

* Career Readiness Score
* Career GPS
* Skill Gap Analysis
* Personalized Learning Roadmap
* Resume Strength Analyzer

## 🧠 Machine Learning

HireSense AI uses Sentence Transformers (`all-MiniLM-L6-v2`) to perform semantic matching between resumes and job descriptions.

Unlike traditional keyword matching, semantic matching understands the meaning behind skills and experiences.
