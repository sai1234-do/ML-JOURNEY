# AI Resume Chatbot

A Retrieval-Augmented Generation (RAG) application built using:

- Streamlit
- Sentence Transformers
- ChromaDB
- FLAN-T5 LLM
- Python

Features:
- Upload Resume (.docx)
- Semantic Search
- AI-powered Question Answering
- Vector Database Retrieval

Tech Stack:
Python, Streamlit, ChromaDB, Transformers, NLP
